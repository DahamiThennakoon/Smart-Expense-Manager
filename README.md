# 💰 Smart Expense Tracker System

A simple and user-friendly web-based Expense Tracker System developed to help users efficiently manage and monitor their daily expenses using CRUD operations.

---

## 📌 Project Overview

The Smart Expense Tracker System is a full-stack web application that allows users to record, update, delete, and view their expenses in an organized way. The system helps users better understand their spending habits and manage their finances effectively.

This project was developed as part of my learning journey to strengthen my skills in web development and database integration.

---

## ✨ Features

- ➕ Add new expense records  
- ✏️ Edit existing expenses  
- ❌ Delete expenses  
- 📋 View all expenses in a table  
- 🏷️ Category-wise expense tracking  
- 📱 Simple and responsive UI  
- 💾 MySQL database integration  
- ✅ Form validation

## ⚙️ How to Run This Project

1. Install XAMPP  
2. Start Apache & MySQL  
3. Copy project folder into `htdocs`  
4. Open phpMyAdmin  
5. Create database (e.g. expense_tracker)  
6. Import `database.sql` file  
7. Open browser and run:

http://localhost/Smart-Expense-Tracker/
